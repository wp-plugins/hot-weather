Hot Weather by HotThemes is a weather forecast widget for WordPress.

== Description ==

"Hot Weather" plugin by [HotThemes](http://www.hotjoomlatemplates.com/free-wordpress-plugins "HotThemes") is a weather forecast widget for WordPress, powered by WeatherLet.com. This plugin downloads data from WeatherLet.com site for the selected location. It's possible to select any city in the USA by ZIP code, or any city in the world by the WeatherLet's city code. The widget can show temperature in Fahrenheit or Celsius degrees. Two forms of the widget are available: a simple static image or rotating Flash based carousel that shows temperature for 7 days upfront.

== Installation ==

1. Install plugin as usually from WordPress.org, or download, and upload `plugin_hot_weather.zip` file your WordPress.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Publish the Hot Weather widget in one of available widget positions.
4. Click on the widget will open all Hot Weather options.
5. Detailed explanation of each plugin option and live demo can be found on [Weather](http://www.hotjoomlatemplates.com/free-wordpress-plugins/206-wordpress-weather-plugin "Weather") page.
